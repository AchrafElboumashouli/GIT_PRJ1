# model/

This directory stores trained ML artifacts generated by the notebook.

## Files generated after running `notebooks/01_preprocessing_training.ipynb`:

| File                  | Description                              |
|-----------------------|------------------------------------------|
| `best_model.pkl`      | Best trained sklearn model (joblib)      |
| `tfidf_vectorizer.pkl`| Fitted TF-IDF vectorizer                 |
| `label_encoder.pkl`   | Label encoder (negative/neutral/positive)|
| `model_metadata.json` | Model metrics & config summary           |

## Usage:

```python
import joblib

model    = joblib.load('model/best_model.pkl')
tfidf    = joblib.load('model/tfidf_vectorizer.pkl')
le       = joblib.load('model/label_encoder.pkl')

text_vec = tfidf.transform(["This product is amazing!"])
pred     = model.predict(text_vec)
label    = le.inverse_transform(pred)[0]
print(label)  # → "positive"
```
